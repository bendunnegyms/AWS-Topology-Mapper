Amazon Web Services Topology Terminology.

Term Defination How to filter for it
EC2 Elastic Compute Cloud is a virtual server in Amazon Web services terminology All graphs display ec2 instances. Single node graph currently lets you search a single ec2 instance by id and see all its in-going and out-going conections
Load Balancer The process of distributing a set of tasks over a set of resources, with the aim of making their overall processing more efficient. App Balancer a type of load balancer is currently visble on all graphs
Data Base AWS supports multiple data bases such as RDS. Click on the data base icon on any graph
App Balancer Short for Application Load Balancer. It is a type of load balancer Click on the app balancer icon on any graph
RDS Relational Database Service (Amazon RDS) is a web service that allows you to quickly create a relational database instance in the cloud. Currently appears on all the graphs in the form of data base icon
Security Groups A security group acts as a virtual firewall for EC2 instances to control incoming and outgoing traffic. In the security groups graph search for a security group and press enter to see its details. Multiple security groups can be searched by listing them and seperating by a comma. Ensure security group is typed exactly with correct casing and spacing .
Source The instance that the current instance is connected to Instance id of the source is listed in the graph link.
Instance Name The instance name given to the specific instance The names of the instances are displayed on the screen. Hover over a specifc link to highlight it and see the direction of the conection.
Ports & Protocols Referes to the ports and protocol connections of an instance. Hover over a graph link to see the ports and protocols assoicated with it.The ports page can also be used to display the connection on specified ports

Graph Overview

Name What is visualises How to use it
Single Node This graph shows the conection between an ec2 instance to all other ec2 instances. Simply press enter in the search bar for both incoming and outgoing. Filter to just incoming by pressing the down arrow on screen or the up arrow for outgoing .Alternativly press the left arrow for out or right arrow for in on the keyboard instead of enter.
Entire Graph Visualisation of all the instance connected to each other. Hover over the links or nodes to find the information
Security Group Shows the connection of single or multiple security group In the security groups graph search for a security group and press enter to see its details. Multiple security groups can be searched by listing them and seperating by a comma. Ensure security group is typed exactly with correct casing and spacing .
Ports Displays the nodes connected to the ports specified. In the security groups graph search for a port and press enter to see its details. Multiple ports can be searched by listing them and seperating by a comma.
